# Rockz 💬

Real-time messaging and calling app.

## Run locally

1. Install Node.js from https://nodejs.org (click the LTS version)
2. Open a terminal and go to this folder
3. Run: npm install
4. Run: npm start
5. Open http://localhost:3000

## Use on your phone too

Find your computer's IP address:
- Windows: open cmd, type `ipconfig`, find "IPv4 Address" (eg. 192.168.1.5)
- Mac/Linux: open Terminal, type `ifconfig | grep "inet "`, find your local IP

Then on your phone (same WiFi): open http://192.168.1.5:3000
